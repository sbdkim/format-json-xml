# Contributing

Thanks for helping improve Format JSON & XML.

## Development flow

1. Install dependencies with `npm install`
2. Run the app locally with `npm run dev`
3. Validate changes with `npm run lint`, `npm run test`, and `npm run build`

## Guidelines

- Keep the app fully static and browser-first
- Preserve the privacy-first product stance
- Add or update tests for formatter behavior changes
- Avoid introducing server dependencies unless the product direction changes
